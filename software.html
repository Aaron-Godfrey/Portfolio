Pharmini

Pharmini is a grant-funded research project at ASU that aims to inspire young adults to think in an industrial engineering mindset with a pharmaceutical context. The software is a city-building style game where players manage a network of randomly generated hospitals by setting up supply chains to help develop medicine for the hospitals.

My Contributions to the project include:

  • Creating interactive context menus that drive gameplay functionality
  • Visually representing backend functions and properties in easy-to-understand formats
  • Adding visual quality of life adjustments to keep gameplay intuitive

NASA Surface Data VR is an ASU and NASA joint project where we were tasked to create a VR experience for the general public to generate excitement and educate users about future space exploration. The software aims to give users a one-of-a-kind experience, simulating various activities on the surface of Mars as well as informing players about the ongoing Psyche mission that inspired this project.

My Contributions to the project include:

  • Designing various activies to make the learning process fun
  • Developing a minigame where the user controls the Mars Curiosity Rover on the Martian surface
  • Curating a main hub that acts as a museum exhibit dedicated to the Psyche mission

This is the model

Arachnoid is a couch co-op arena shooter game where players take control of tanks with various loadouts and try to defeat each other. Developed over the course of four months, the project served as the first iteration of an ASU summer course where students would come together to form a game studio over the course of 4 months.

My Contributions to the project include:

  • Developing custom player controllers for each tank type to handle physics and movement abilities
  • Handled coding framework for adding/storing gameplay/weapon modifiers on each character playing
  • Creating Menu and Lobby scenes with loadout customization options
