This is the model

Pharmini is a grant-funded research project at ASU that aims to inspire young adults to think in an civil engineering mindset through a pharmaceutical context. The software is a city-building style game where players manage a network of randomly generated hospitals and fulfill their needs by building nearby factories and labs to produce materials and orders.

My Contributions to the project include:

  • Creating interactive context menus to carry out actions for a given building
  • Visually representing backend functions and properties in easy-to-understand formats
  • Adding quality of life adjustments to keep gameplay intuitive
  • Designed Glossary tab to help onboard new players

This is the model

NASA Surface Data VR is an ASU and NASA joint project where students were tasked to create a VR experience for the general public to generate excitement and educate users about future space exploration. The educational program aims to give users a one-of-a-kind experience, simulating various activities on the surface of Mars as well as informing players about the ongoing Psyche mission that inspired this project.

My Contributions to the project include:

  • Designing various activies to make the learning process fun
  • Developing a minigame where the user controls the Curiosity Rover on the Martian surface
  • Curating a main hub that acts as a museum exhibit dedicated to the Psyche mission

  • SCRUM
  • External API Integration

This is the model

Arachnoid is a couch co-op arena shooter where players take control of tanks with various loadouts and try to defeat each other. Developed over the course of four months, the project served as the first iteration of an ASU summer course where students would come together to form a game studio and create a prototype .

My Contributions to the project include:

  • Developing custom player controllers for each tank type to handle physics and movement abilities
  • Handled coding framework for adding/storing gameplay/weapon modifiers on each character playing
  • Creating Menu and Lobby scenes with loadout customization options
